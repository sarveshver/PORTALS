MACHINE LEARNING

The project aims to enhance cybersecurity by developing a system for predicting phishing websites using machine learning algorithms, specifically Random Forest and k-Nearest Neighbors (KNN). The process involves collecting a dataset with labeled examples of phishing and legitimate websites, preprocessing the data, and extracting informative features related to URL structure, content characteristics, and domain information.The models are evaluated using standard metrics, including accuracy, precision, recall, and F1 score, to ensure their effectiveness in generalizing to new, unseen data. Once validated, the models can be deployed in a cybersecurity system, automatically identifying and flagging potentially malicious websites. Continuous monitoring and periodic updates to the models are crucial to adapt to evolving phishing techniques and enhance the system's overall efficacy in detecting emerging threats.

IOT

The IoT Surveillance Cam Robot is a smart, remote-controlled robot designed for monitoring and surveillance applications. This project employs the use of ESP32 and ESP8266 microcontrollers, along with the L298N motor driver, to create a versatile and connected robotic system. The ESP32 serves as the main control unit and camera module, providing high-quality imaging capabilities. It connects to a Wi-Fi network, enabling seamless communication with the surrounding environment. The ESP8266, integrated with the L298N motor driver, controls the robot's movements in response to commands received via Wi-Fi.

IOT

The Home Automation System using ESP-32 (Rower) is designed to provide convenient control over household electronic devices. Utilizing the ESP-32 microcontroller, this project offers a robust and efficient solution for home automation. Integration with the Google API enhances user experience by enabling control through a dedicated mobile application. ESP-32 serves as the central hub for connecting and controlling electronic devices within the home. The use of the Rower library enhances the capabilities of ESP-32, making it a powerful tool for managing various appliances and systems. The Google API integration allows users to control and monitor their home devices remotely using a mobile application. The application provides an intuitive interface for users to interact with and customize their home automation settings. Through the Google API, users can issue voice commands, schedule tasks, and receive real-time updates on the status of connected devices. Key features include the ability to control lighting, temperature, security systems, and other electronic appliances through the ESP-32 microcontroller.

The Web Application developed for this project leverages the Django framework to curate and present real-time news data from over 10 top news websites. By integrating the News API, the application aggregates news articles, headlines, and updates, offering users a centralized platform to stay informed. Using Django as the web framework provides a robust structure for managing the backend, handling data processing, and rendering dynamic web pages. The application seamlessly connects to the News API, fetching and organizing news content from a diverse range of reputable sources. Key features of the web app include a user-friendly interface for navigating through news categories, personalized preferences, and a visually appealing display of articles. The integration with the News API ensures that the information presented is up-to-date and relevant.
